### Persona-Specific Task Generation — Fullstack Engineer

Focus task breakdown on:
- **End-to-End Features**: Each feature spans frontend component + API endpoint + database
- **API Layer**: Contract tests, endpoint implementation, request/response validation
- **Frontend Layer**: Component creation, state management, API integration
- **Data Flow**: Database schema, API response, frontend state, UI rendering
- **Integration Tasks**: Frontend-backend integration tests, contract alignment validation
- **Type Safety**: Shared type definitions, API contract generation, frontend type imports
- **Error Handling**: Backend error responses, frontend error handling, user-friendly messages
- **Performance**: API optimization, database query tuning, frontend bundle size, lazy loading
- **Security**: Backend authentication/authorization, frontend route guards, XSS prevention
- **Testing Strategy**: Contract tests, backend integration tests, frontend component tests, E2E tests
- **Dependencies**: Database schema → API → types → frontend component → integration

Break down features into full-stack subtasks:
- Database schema subtask
- Backend model/repository subtask
- API endpoint subtask
- API client/types subtask
- Frontend component subtask
- Integration test subtask
