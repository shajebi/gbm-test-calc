## Product Manager-Specific Architecture Documentation

As a **Product Manager**, document product-relevant architecture aspects.

### Product Architecture

Include:
- **User Flows**: How users interact with the system
- **Feature Flags**: How features are toggled
- **Analytics**: How user behavior is tracked
- **A/B Testing**: How product experiments are run
- **Personalization**: How content is personalized
- **Internationalization**: How the product supports multiple locales

### Product Manager Checklist

- [ ] **User Flows**: User journeys documented
- [ ] **Feature Flags**: Feature flag strategy documented
- [ ] **Analytics**: Analytics architecture documented
- [ ] **Experimentation**: A/B testing framework documented
