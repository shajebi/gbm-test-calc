### QA Engineer Fact-Checking Standards

**Philosophy**: Testing standards and framework compatibility must be verified with authoritative sources to ensure reliable test strategies.

---

#### Critical Claim Types

**1. Testing Standards** (Required Quality: A, 90%+)
- ISTQB standards
- IEEE testing standards
- Industry best practices

**Verification Requirements**:
- Tier 1 standards bodies only (IEEE, ISTQB, ISO)
- Official documentation required
- Current version verification

**2. Framework Compatibility** (Required Quality: B+, 85%+)
- Test framework versions
- Browser/platform support
- Integration capabilities

**Verification Requirements**:
- Official framework documentation
- Verified compatibility matrices
- Recent release notes (last 6 months)

**3. Test Coverage Benchmarks** (Required Quality: B+, 85%+)
- Industry coverage standards
- Performance benchmarks
- Reliability metrics

**Verification Requirements**:
- Industry reports or surveys
- Published benchmarks
- Peer-reviewed studies

---

#### Integration with QA Workflow


**Note**: Framework compatibility claims get extra verification assistance.
