## QA Engineer-Specific Architecture Documentation

As a **QA Engineer**, document testing architecture and quality assurance.

### Testing Architecture

Include:
- **Test Environments**: Test, staging, production environments
- **Test Data Management**: How test data is created and managed
- **Test Automation**: Test automation frameworks and tools
- **Performance Testing**: Load testing, stress testing infrastructure
- **Security Testing**: Security testing tools and processes
- **Test Reporting**: How test results are reported

### QA Engineer Checklist

- [ ] **Test Strategy**: Overall testing strategy documented
- [ ] **Test Environments**: Test environment architecture documented
- [ ] **Test Automation**: Automation framework documented
- [ ] **Test Data**: Test data management documented
