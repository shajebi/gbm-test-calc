Build a full-stack calculator application with a simple one-page HTML + CSS frontend and a Python backend.

Frontend
	•	Single-page HTML with vanilla CSS (no React).
	•	Clean, minimal calculator layout.
	•	Support dark and light mode with a toggle (CSS variables).
	•	Buttons for basic operations: +, -, *, /.
	•	Include memory functions: M+, M-, MR, MC.
	•	Use vanilla JavaScript to handle UI logic and call backend APIs.

Backend
	•	Python FastAPI backend.
	•	Dependency management using uv.
	•	REST endpoints for:
	•	Performing calculations
	•	Managing calculator memory
	•	Memory stored per session (in-memory is acceptable).

General
	•	Simple, readable project structure.
	•	Minimal setup and run instructions.
	•	Focus on clarity, correctness, and maintainability.
